# Flatsweep

Flatpak leftover cleaner

<img src="https://raw.githubusercontent.com/giantpinkrobots/flatsweep/main/screenshots/Screenshot-Flatsweep-1.png" style="zoom:33%;" />

When you uninstall a Flatpak, it can leave some files behind on your computer. Flatsweep helps you easily get rid of the residue left on your system by uninstalled Flatpaks.

## Building

Clone this repository and open it using GNOME Builder. Then, you can build and install it onto your system.
